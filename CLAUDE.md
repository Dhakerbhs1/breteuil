# Commands
- Dev: `npm run dev`
- Build: `npm run build`
- Start: `npm run start`

# Project Context
- **Framework**: Next.js 16 (App Router)
- **Styling**: Vanilla CSS Modules (no Tailwind)
- **Identity**: "Warm Minimal Luxury" (Charcoal, Terracotta, Cream, Gold)
- **Typography**: Playfair Display (Serif), Jost (Sans-serif)

# Design System Rules
- Do NOT use blue, green, or bright red (reserved for other brands like Beauchamp/Degrassi).
- Maintain "narrative scrolling" layouts over basic grid blocks.
- Interactions should be smooth, slow reveals (0.8s+ transitions).

# Editorial Rules
- **Prohibited**: Non-verifiable stats ("98% satisfaction"), promises of "zéro casse", specific experience years (unless provable), fake reviews.
- **Formules**: Strictly "Économique", "Confort", "Luxe".
- **Mentions légales**: Exact wording strictly required. Vercel is the host, SIRET and Head Office placeholders to be filled. No CNIL links.
